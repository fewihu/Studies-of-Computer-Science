# Kasiski-Test
* Implementierung des Kasiski-Test zur Schlüssellängenbestimmung
* findet gleiche Folgen von Chiffratzeichen und bestimmt ihren Abstand
* Primfaktoren der Abstände sind Hinweis auf Schlüssellänge Vigenere-verschlüsselter Chiffrate
* Übersetzen mit: gcc -o kasiski3 kasiski3.c -Wall -lm -pthread
